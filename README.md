# Employee Salary Analysis Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes employee salary data to provide insights into workforce distribution and salary trends across departments and cities.
![Uploading Dashboard.png…]()

## 🛠️ Tools Used
- Power BI
- Microsoft Excel

## 📊 Dashboard Features
- Total Employees
- Average Salary
- Maximum Salary
- Minimum Salary
- Average Salary by Department
- Employee Count by City
- Total Salary by Department
- Department & City Slicers

## 💼 Business Insights
- Compare average salary across departments.
- Identify cities with the highest employee count.
- Analyze department-wise salary distribution.
- Filter data interactively using slicers.
